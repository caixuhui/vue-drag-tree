# vue-drag-tree
这是一个操作json数据的树形组件，可以拖拽把节点移动到父节点或者子节点。

基于`Vue2.x` 和`Vuex`的进行开发，因为有点懒所以组件内的按钮和图标是使用了[Element-UI](https://github.com/ElemeFE/element "Element-UI")的组件。

代码目录是`vue-cli`的自动生成目录，直接运行`npm install` 就可以运行起来。

目前版本还只是雏形，后续会继续开发完善...
